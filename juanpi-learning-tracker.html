📋 Conceptos clave

La terminal es una interfaz de texto para hablar con el ordenador. Claude Code vive aquí — no tiene botones ni menús, solo comandos.

Qué quieres hacerWindows (tu caso)Mac / Linux
¿Dónde estoy?cdpwd
¿Qué hay aquí?dirls
Entrar a una carpetacd NombreCarpetacd NombreCarpeta
Volver atráscd ..cd ..
Limpiar pantallaclsclear
Crear carpetamkdir nombremkdir nombre
💡 Lo más importante: Claude Code no es un chatbot web — vive en la terminal y actúa directamente sobre tus archivos.

CLAUDE.md: darle contexto a Claude
El fichero de instrucciones · cómo personalizarlo · ejemplos
teoría práctica 5/5 ⭐
▼
📋 Conceptos clave

CLAUDE.md es un fichero de texto en la raíz del proyecto que Claude Code lee automáticamente al arrancar. Es el briefing permanente del proyecto.

SecciónPara qué sirve
ContextoQué es el proyecto, quién lo usa, para qué
NUNCA tocarFicheros críticos, backups, datos sensibles
ConvencionesNombres de fichero, idioma, estructura
Tareas frecuentesAtajos para cosas que pides a menudo
Dónde va: en la carpeta raíz del proyecto, junto al resto de ficheros.
Formato: texto plano en Markdown. Sin sintaxis especial obligatoria.
💡 Puedes escribirlo tú o pedirle al propio Claude Code que lo genere describiendo tu proyecto.

📋 Conceptos clave

Claude Code puede leer múltiples ficheros, procesar datos y generar informes ejecutivos completos con una sola instrucción en lenguaje natural.

Flujo de trabajoCómo hacerlo
Pedir el plan primeroDentro de la sesión: "explícame qué vas a hacer, no ejecutes nada"
Dar el OK"Perfecto, adelante" tras revisar el plan
Pedir mejorasVolver a entrar y pedir ajustes conversacionalmente
Retomar sesiónclaude --resume [id]
--plan: flag obsoleto en versiones recientes. Alternativa: pedir el plan conversacionalmente dentro de la sesión interactiva.
💡 CLAUDE.md actúa en segundo plano — Claude ya sabe en qué proyecto está sin que se lo expliques cada vez.

Glosario esencial
Terminal / CLI
Interfaz de texto para hablar con el ordenador. La base de Claude Code.
Claude Code
Herramienta de Anthropic que vive en tu terminal. Lee tu código y actúa directamente sobre tus archivos.
npm / Node.js
El "motor" necesario para instalar Claude Code. Node.js es el entorno; npm es su gestor de paquetes.
Agente IA
IA que no solo responde preguntas, sino que ejecuta acciones de forma autónoma en cadena.
MCP
Model Context Protocol. El protocolo que permite a Claude conectarse con herramientas externas (email, Drive, APIs…).
Orquestador
Sistema que coordina múltiples agentes o pasos automáticos. Ejemplos: n8n, Power Automate.
